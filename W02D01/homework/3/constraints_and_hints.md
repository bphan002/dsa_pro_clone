# Number of Sub-arrays of Size K and Average Greater than or Equal to Threshold Constraints and Hints

## Constraints
-   `1 <= arr.length <= 10^5`
-   `1 <= arr[i] <= 10^4`
-   `1 <= k <= arr.length`
-   `0 <= threshold <= 10^4`

## Hint #1
> Does this problem concern two elements or the contents between those elements?
